기능 파이썬 코드/메서드 시간 복잡도 비고  
끝에 추가 append() O(1) JS의 push()와 동일  
끝에서 제거 pop() O(1) JS의 pop()과 동일  
앞에서 제거 pop(0) O(n) JS의 shift()와 동일. 인덱스 한 칸씩 당겨짐  
앞에 추가 insert(0, x) O(n) JS의 unshift()와 동일. 인덱스 밀려남  
중간 삽입/삭제 insert(i, x), del list[i] O(n) JS의 splice() 역할  
정렬 sort() O(n \log n) Timsort 알고리즘 사용  
리스트 연결 list1 + list2, extend() O(k) 연결하는 리스트의 길이에 비례  
슬라이싱(복사) list[a:b] O(k) 복사하는 요소 개수 k에 비례  
요소 검색 index(), in 연산자 O(n) 처음부터 끝까지 순회  
문자열 연결 ''.join(list) O(n) JS의 join()과 동일  
문자열 분리 split() O(n) JS의 split()과 동일  
특정 문자 접근 string[i] O(1) JS의 charAt()과 동일
